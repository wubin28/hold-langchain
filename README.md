# Hold LangChain 示例

## Quickstart for learn-AI-app-dev-from-scratch/langchain_v1_fewshot_example.py on macOS

```
cd learn-AI-app-dev-from-scratch

python3 -m venv .venv

source .venv/bin/activate

pip install --upgrade pip

pip install -r requirements.txt

python langchain_v1_fewshot_example.py

deactivate
```
